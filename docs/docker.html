An Introduction to Docker

  • Docker is a virtualization platform that allows you to run sandboxed applications on a host operating system.
  • Docker follows the same idea of virtualization, but operates very differently.
  • Docker is container based technology and containers are just user space of the operating system.
  • A Dockerized application is just a process that runs on your system.

Containers

  • A container is a standard unit of software that packages up code and all its dependencies so the application runs quickly and reliably from one computing environment to another.
  • A Docker container image is a lightweight, standalone, executable package of software that includes everything needed to run an application: code, runtime, system tools, system libraries and settings.

Images

  • Images become containers when they run on Docker Engine.
  • A Docker container image is a lightweight, standalone, executable package of software that includes everything needed to run an application: code, runtime, system tools, system libraries and settings.
